# MathType Crack v1.0

简单版 MathType 破解脚本，自动关闭 MathType、删除注册表限制、重启程序。

## 使用方法

1. 双击 `mathtype-crack.bat`
2. 等待脚本自动执行
3. 完成

## 文件说明

| 文件 | 说明 |
|------|------|
| `mathtype-crack.bat` | 主程序入口 |
| `mathtype-crack.ps1` | 脚本源码 |

## 适用场景

- 单台电脑使用
- 路径已固定的情况

## 版本区别

- **v1.0**: 简单版，路径硬编码
- **v2.0**: 配置版，首次运行需设置一次，之后通用
